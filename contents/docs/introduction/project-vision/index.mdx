## 🌍 Visión del Proyecto

### 📌 ¿Qué es esta herramienta?

La **Herramienta de Perspectivas sobre una JET en AFOLU** es una plataforma diseñada para ayudar a identificar oportunidades y desafíos en la **transición justa y equitativa** dentro del sector **AFOLU (Agriculture, Forestry, and Other Land Use)**. Su enfoque está en proporcionar una evaluación clara y estructurada para proyectos de mitigación y adaptación al cambio climático en **América Latina y el Caribe**.

Los usuarios pueden evaluar sus proyectos a través de un sistema guiado de preguntas y respuestas, obteniendo métricas clave como el **Just Transition Indicator (JTI)**, que permite medir qué tan alineado está un proyecto con los principios de transición justa.

### 🚀 Visión tecnológica

Desde el punto de vista tecnológico, el sistema ha sido diseñado para manejar grandes volúmenes de datos sin comprometer la **escalabilidad, el rendimiento ni la estabilidad**.

📌 **Objetivos tecnológicos clave:**
✅ **Soporte para Big Data:** La arquitectura permite almacenar y procesar grandes volúmenes de información sin afectar el rendimiento del sistema.
✅ **Alta disponibilidad:** La infraestructura en la nube asegura que la plataforma esté siempre accesible, sin interrupciones.
✅ **Baja latencia en consultas:** El acceso a los datos está optimizado con un modelo basado en queries eficientes en bases de datos NoSQL.
✅ **Escalabilidad automática:** El sistema crece de manera dinámica según la demanda, sin necesidad de intervención manual.

### 🏗️ Fundamentos arquitectónicos

Para cumplir con estos objetivos, la plataforma se basa en una combinación de **tecnologías modernas y arquitecturas optimizadas**:

🔹 **AWS CDK + DynamoDB** → Permite una infraestructura flexible, con una base de datos optimizada para consultas rápidas sin necesidad de escaneos completos.
🔹 **Next.js + Vercel** → Asegura una experiencia de usuario fluida, con despliegues instantáneos y optimización automática de carga.
🔹 **Microservicios y APIs desacopladas** → Garantizan que cada componente pueda escalar de forma independiente, evitando cuellos de botella.

Esta combinación tecnológica permite que la herramienta pueda **manejar grandes cantidades de datos sin afectar el rendimiento**, asegurando que los usuarios siempre tengan acceso a información actualizada y en tiempo real.

📌 **En resumen, esta plataforma no solo ofrece una evaluación robusta de proyectos en transición justa, sino que lo hace con una infraestructura de alto rendimiento y preparada para el futuro.**
